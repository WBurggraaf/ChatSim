# ChatSim

Static GenAI chat simulator for presentations.

## Usage

1. Open [chatsim.html](/C:/Users/posit/ChatSim/chatsim.html) in a browser.
2. Press `Start Demo`.
3. Use `Space` or `Right Arrow` to advance.
4. Use `R` to restart.

## What it includes

- Staged user and assistant messages
- Typing animation for assistant replies
- Full-screen popup moments for presenter emphasis
- Sidebar step navigation
- Progress indicator and restart flow

## Customization

Edit the `chatData` object near the top of [chatsim.html](/C:/Users/posit/ChatSim/chatsim.html) to change:

- Demo title
- Model label
- Step content
- Timing values

No model downloads or external runtime dependencies are required.
